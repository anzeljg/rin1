9.

Obrnjeni poljski zapis (RPN) je zapis, ki nam omogoča računanje aritmetičnih izrazov brez oklepajev. Izraz, zapisan v RPN, je zaporedje števil (omejili se bomo na cela števila med 0 in 9) in aritmetičnih operatorjev (omejili se bomo na operatorje +, - in *). Vrednost izraza v RPN izračunamo tako, da najprej ustvarimo prazen sklad, nato pa se pomikamo po znakih izraza. Ko naletimo na število, ga dodamo na sklad. Ko naletimo na operator, odvzamemo dve števili s sklada (recimo, da smo najprej odvzeli število a, potem pa število b), izračunamo vrednost izraza b + a, b − a oziroma b · a (odvisno od operatorja) in rezultat dodamo na sklad. Če je izraz pravilno zapisan, bo ob koncu postopka na skladu ostal en sam element. To je rezultat izraza.
Na primer, vrednost izraza 345+*6- znaša 21. V običajnem zapisu bi ta izraz izgledal takole: 3 ∗ (4 + 5) − 6.

Napiši funkcijo IzracunajRPN(rpn), ki vrne vrednost podanega izraza, zapisanega v obliki RPN. Lahko predpostaviš, da niz rpn vsebuje samo števke od 0 do 9 in znake +, - in *, in to v veljavnem zaporedju.

import sklad as sk

# Vrne rezultat izraza, zapisanega v
# obliki RPN.
def IzracunajRPN(rpn):
sklad = sk.Ustvari()

# Sprehodi se po znakih izraza.
for znak in rpn:
if znak >= '0' and znak <= '9':
# Trenutni znak je števka.
stevilo = ord(znak) - ord('0')
sk.Dodaj(sklad, stevilo)

else:
# Trenutni znak je operator.
a = sk.Odvzemi(sklad)
b = sk.Odvzemi(sklad)
if znak == '+':
sk.Dodaj(sklad, b + a)
elif znak == '-':
sk.Dodaj(sklad, b - a)
else:
sk.Dodaj(sklad, b * a)

# Če je izraz pravilno zapisan, na
# skladu ostane eno samo število.
# To število je rezultat izraza.
return sk.Odvzemi(sklad)

# Preverimo delovanje funkcije.
print(IzracunajRPN('345+*6-'))
print(IzracunajRPN('345--'))
print(IzracunajRPN('34-5-'))
print(IzracunajRPN('7'))

10.

Napiši funkcijo PretvoriVRPN(izraz), ki vrne pretvorbo podanega običajno zapisanega izraza v zapis RPN. Izraz je sestavljen samo iz enomestnih pozitivnih števil ter operatorjev +, - in *. Presledkov in oklepajev ne vsebuje.
Na primer, pretvorba izraza 3+4*5-6 je enaka 345*+6-.

Če izraz ne vsebuje nič drugega kot eno samo števko, potem je njegova pretvorba v zapis RPN kar enaka samemu izrazu. V nasprotnem primeru pa najprej poiščemo najbolj desni najšibkejši operator o (to je + ali -, če teh ni, pa poiščemo najbolj desni operator *), nato pa zapis RPN sestavimo kot ABo, kjer je A zapis RPN, ki ga pridobimo iz podizraza levo od operatorja o, B pa zapis RPN, ki ga pridobimo iz podizraza desno od operatorja o. Zapisa A in B pridobimo po enakem postopku kot zapis za celoten izraz.

# Vrne pretvorbo podanega običajno zapisanega
# izraza v zapis RPN.
def PretvoriVRPN(izraz):

# Če je dolžina izraza enaka 1, potem
# izraz po predpostavki vsebuje samo eno
# število, zato je njegova pretvorba v
# RPN kar enaka samemu izrazu.
if len(izraz) == 1:
return izraz

# Sicer poiščemo indeks najbolj desnega
# najšibkejšega operatorja (indno).

# Najprej poskusimo s plusom in minusom.
indno = -1
for i in range(len(izraz) - 1, -1, -1):
if izraz[i] == '+' or izraz[i] == '-':
indno = i
break

# Če izraz ne vsebuje niti plusa niti
# minusa, poiščemo najbolj desni
# operator množenja.
if indno < 0:
for i in range(len(izraz) - 1, -1, -1):
if izraz[i] == '*':
indno = i
break

# V RPN pretvorimo del izraza pred
# najbolj desnim najšibkejšim
# operatorjem.
leviRPN = PretvoriVRPN(izraz[: indno])

# V RPN pretvorimo del izraza za najbolj
# desnim najšibkejšim operatorjem.
desniRPN = PretvoriVRPN(izraz[indno + 1 :])

# Sestavimo končni izraz.
return leviRPN + desniRPN + izraz[indno]

# Preverimo delovanje funkcije.
print(PretvoriVRPN('3'))
print(PretvoriVRPN('3+4*5-6'))
print(PretvoriVRPN('3*4+5*6'))
print(PretvoriVRPN('3-4+5*6'))
print(PretvoriVRPN('3-4-5-6'))
