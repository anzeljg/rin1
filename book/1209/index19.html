4.

V samostojnem programu (ne v modulu sklad.py) napiši funkcijo ObrniSklad(sklad), ki vrne sklad, v katerem so elementi zapisani v obratnem vrstnem redu kot v podanem skladu. Na primer, če funkcijo pokličemo nad skladom ⟨10, 20, 30⟩, dobimo sklad ⟨30, 20, 10⟩. Funkcija lahko spreminja podani sklad, uporabljati pa sme samo funkcije iz modula sklad.py.

Elemente preloži iz podanega sklada v nov sklad.

import sklad as sk

# Vrne obrnjeno kopijo podanega sklada.
def ObrniSklad(sklad):
noviSklad = sk.Ustvari()
# Naloga je enostavna: elemente
# odvzemamo s sklada in jih sproti
# dodajamo na novi sklad. Na novem
# skladu bodo elementi shranjeni v
# obratnem vrstnem redu.
while not sk.JePrazen(sklad):
element = sk.Odvzemi(sklad)
sk.Dodaj(noviSklad, element)
return noviSklad

# Preverimo delovanje funkcije.
sklad = sk.Ustvari()
a = [5, 2, 5, 5, 4, 9, 10]
for e in a:
sk.Dodaj(sklad, e)
sk.Izpisi(ObrniSklad(sklad))

5.

Napiši program, ki v zanki bere cela števila. Ob vnosu pozitivnega števila naj ne izpiše ničesar, ob vnosu negativnega števila pa naj izpiše prvo izmed doslej prebranih pozitivnih števil, ki ga še ni izpisal. Če je program izpisal že vsa doslej vnešena pozitivna števila, naj se ne zgodi nič. Program naj se zaključi, ko uporabnik vnese število 0 (glej primer).

Pomagaj si z vrsto.

# Program v zanki bere cela števila. Ob
# vnosu negativnega števila izpiše prvo
# izmed doslej prebranih pozitivnih števil,
# ki ga še ni izpisal.

import vrsta as vr

# Delali bomo z vrsto.
vrsta = vr.Ustvari()
stevilo = int(input('Vnesi število: '))

while stevilo != 0:
if stevilo > 0:
# Če je število pozitivno, ga dodamo
# v vrsto.
vr.Dodaj(vrsta, stevilo)

else:
# Če je število negativno (in če vrsta
# ni prazna), odvzamemo število iz
# vrste in ga izpišemo. To bo ravno
# prvo pozitivno število, ki ga še
# nismo izpisali.
if not vr.JePrazna(vrsta):
print(vr.Odvzemi(vrsta))

stevilo = int(input('Vnesi število: '))

6.

Napiši program, ki v zanki bere pozitivna cela števila in sproti izpisuje vsoto doslej vnešenih, vendar omogoča uporabniku, da z vnosom negativnega celega števila razveljavi zadnji vnos pozitivnega števila. Pomagaj si s skladom.

# Program, ki v zanki bere pozitivna cela
# števila in izpisuje vsoto doslej vnešenih,
# pri tem pa omogoča tudi razveljavitve
# (z vnosom negativnega celega števila).

import sklad as sk

# Delali bomo s skladom.
sklad = sk.Ustvari()
stevilo = int(input('Vnesi število: '))
vsota = 0

while stevilo != 0:
if stevilo > 0:
# Če je število pozitivno, ga dodamo
# na sklad.
sk.Dodaj(sklad, stevilo)
vsota += stevilo

else:
# Če je število negativno (in če sklad
# ni prazen), odvzamemo število s
# sklada in ga odštejemo od vsote.
if not sk.JePrazen(sklad):
razveljavljeno = sk.Odvzemi(sklad)
vsota -= razveljavljeno
print('Razveljavil sem število',
str(razveljavljeno))

print('Vsota znaša', str(vsota))
stevilo = int(input('Vnesi število: '))
