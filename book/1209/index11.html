Rešitev

Program za merjenje porabe časa: 

from datetime import datetime
import povezaniSeznam as sez

K = 10000

# Izmerimo porabo časa za K, 2*K, 3*K, ..., 10*K vstavljanj na začetek
# povezanega seznama.
for i in range(1, 11):
  # Izračunamo število elementov, ki jih bomo vstavili na začetek.
  n = i * K

  # Ustvarimo seznam.
  ps = sez.Ustvari()

  # n-krat vstavimo isto število (recimo 42) na začetek seznama.
  tik = datetime.now()
  for i in range(n):
    sez.DodajNaZacetek(ps, 42)
  tak = datetime.now()

  # Izračunamo porabo časa v milisekundah.
  cas = (tak - tik).total_seconds() * 1000
  print('Poraba časa za ' + str(n) + ' vstavljanj na začetek: ' +
        str(cas) + ' milisekund')

# Izmerimo porabo časa za K, 2*K, 3*K, ..., 10*K vstavljanj na konec
# povezanega seznama.
print()
for i in range(1, 11):
  # Izračunamo število elementov, ki jih bomo vstavili na začetek.
  n = i * K

  # Ustvarimo seznam.
  ps = sez.Ustvari()

  # n-krat dodamo isto število (recimo 42) na konec seznama.
  tik = datetime.now()
  for i in range(n):
    sez.DodajNaKonec(ps, 42)
  tak = datetime.now()

  # Izračunamo porabo časa v milisekundah.
  cas = (tak - tik).total_seconds() * 1000
  print('Poraba časa za ' + str(n) + ' vstavljanj na začetek: ' +
        str(cas) + ' milisekund')

*Implementacija s povezanim seznamom

Modul

Vse funkcije združimo v modul povezaniSeznam.py:

Izvedi Počisti 



					

					

						



Če modul poženemo kot samostojen program, se bo poklicala funkcija main, ki ustvari povezani seznam in na njem preizkusi posamezne funkcije. Modul vsebuje tudi pomožne funkcije Zaporedje, ZaporedjeNazaj in IzpisiVse, ki olajšajo preverjanje delovanja glavnih funkcij. Funkcija Zaporedje, na primer, vrne tabelo elementov povezanega seznama, v kateri so elementi zapisani po naraščajočih položajih.



Z izjemo pomožnih funkcij Zaporedje, ZaporedjeNazaj in IzpisiVse je časovna zahtevnost funkcij v modulu povezaniSeznam.py neodvisna od števila elementov v povezanem seznamu, torej O(1). Časovna zahtevnost pomožnih funkcij pa seveda znaša O(n), saj se moramo pri vseh treh po sprehoditi po celotnem seznamu.



Vaja
