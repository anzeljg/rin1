Naloge

Sledeče naloge pogosto zahtevajo samo zapis ene ali več funkcij. Vse funkcije, ki jih boš napisal(-a), temeljito preizkusi, tako da jih kličeš z različnimi nabori parametrov.

1.

Napiši funkcijo Poisci(tabela, element), ki vrne indeks prve pojavitve podanega elementa v podani tabeli. Če iskanega elementa v tabeli ni, naj funkcija vrne vrednost −1. Kakšna je povprečna časovna zahtevnost te funkcije?

Povprečna časovna zahtevnost funkcije je enaka O(n).
# Vrne položaj prve pojavitve podanega
# elementa v podani tabeli oziroma -1,
# če elementa v tabeli ni.
def Poisci(tabela, element):

for i in range(len(tabela)):
if tabela[i] == element:
return i

# če pridemo do tod, vemo, da
# elementa ni
return -1

# Preizkusimo našo funkcijo.
print(Poisci(['Z', 'A', 'R', 'J', 'A'], 'A'))
print(Poisci(['Z', 'A', 'R', 'J', 'A'], 'J'))
print(Poisci(['Z', 'A', 'R', 'J', 'A'], 'K'))

2.

Napiši funkcijo Vsebuje(tabela, element), ki vrne True natanko v primeru, če podana naraščajoče urejena tabela vsebuje podani element.

Nalogo lahko rešimo na podoben način kot prejšnjo, precej učinkovitejšo rešitev pa dobimo z uporabo bisekcije:
# Vrne True natanko v primeru, če podana
# naraščajoče urejena tabela vsebuje podani
# element.

def Vsebuje(tabela, element):
# Pomagamo si z bisekcijo. Najprej
# preverimo sredinski element tabele.
# Če je enak iskanemu, vrnemo True.
# Če je iskani element večji od
# sredinskega, na enak način iščemo
# v desni polovici tabele. Če je
# iskani element manjši od sredinskega,
# na enak način iščemo v levi
# polovici tabele.

prviIndeks = 0
zadnjiIndeks = len(tabela) - 1

while prviIndeks <= zadnjiIndeks:
sredina = (prviIndeks + zadnjiIndeks) // 2
if element == tabela[sredina]:
# Našli smo element!
return True

elif element > tabela[sredina]:
# Iskani element je večji od
# sredinskega, zato bomo odslej
# iskali samo v podtabeli
# tabela[sredina + 1 : zadnjiIndeks]
prviIndeks = sredina + 1

else:
# Iskani element je manjši od
# sredinskega, zato bomo odslej
# iskali samo v podtabeli
# tabela[prviIndeks : sredina + 1]
zadnjiIndeks = sredina - 1

# če pridemo do tod, vemo, da elementa ni
return False

# Preizkusimo našo funkcijo.
print(Vsebuje([20, 30, 40, 50, 60, 70], 50))
print(Vsebuje([20, 30, 40, 50, 60, 70], 20))
print(Vsebuje([20, 30, 40, 50, 60, 70], 70))
print(Vsebuje([20, 30, 40, 50, 60, 70], 10))
print(Vsebuje([20, 30, 40, 50, 60, 70], 80))
print(Vsebuje([20, 30, 40, 50, 60, 70], 45))

3.

Napiši funkcijo Prestej(povSeznam, element), ki vrne število pojavitev podanega elementa v podanem povezanem seznamu. Funkcijo napiši v samostojnem programu, ne v modulu povezaniSeznam.py. Modul moraš torej v svoj program uvoziti.

Sprehodi se po seznamu od začetnega vozlišča proti končnemu.

import povezaniSeznam as sez

# Vrne indeks prvega vozlišča, ki vsebuje
# podani element.
def Prestej(povSeznam, element):
stevec = 0

# Pričnemo v začetnem vozlišču, nato pa
# potujemo po verigi vozlišč in štejemo
# elemente.
i = sez.IndeksZacetnega(povSeznam)
while i != 0:
if sez.Element(povSeznam, i) == element:
stevec += 1
i = sez.IndeksNaslednika(povSeznam, i)

# Elementa nismo našli.
return stevec

# Preizkusimo funkcijo.
ps = sez.Ustvari()
sez.DodajNaZacetek(ps, 'A')
sez.DodajNaKonec(ps, 'A')
sez.DodajNaZacetek(ps, 'A')
sez.DodajNaZacetek(ps, 'B')
sez.DodajNaKonec(ps, 'B')
sez.DodajNaZacetek(ps, 'C')

print(Prestej(ps, 'A'))
print(Prestej(ps, 'B'))
print(Prestej(ps, 'C'))
print(Prestej(ps, 'D'))

sez.IzlociKoncnega(ps)
print(Prestej(ps, 'A'))
print(Prestej(ps, 'B'))
print(Prestej(ps, 'C'))
print(Prestej(ps, 'D'))
