Napiši program, ki prebere dve letnici a in b in izpiše skupno število dni v vseh letih od vključno leta a do vključno leta b. (Spomnimo se, da je leto n prestopno natanko tedaj, ko je n deljiv s 400 ali pa je deljiv s 4 in ne s 100.)

a = int(input("Vpiši letnico:")) b = int(input("Vpiši letnico:")) skupajDni = 0 for n in range(a, b + 1): skupajDni += 365 if n % 400 == 0 or n % 4 == 0 and n % 100 != 0: skupajDni += 1 # prestopni dan v letu n print("Vseh dni: " + str(skupajDni))
