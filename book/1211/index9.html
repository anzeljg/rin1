Namig

Množica oseb na razdalji r so vse osebe, s katerimi obogatimo množico dosegljive v koraku r, če obogatitev začetne prazne množice štejemo kot korak 0.

Rešitev

# Vrne množico indeksov vseh oseb, ki so od osebe z indeksom 'oseba' po
# poznanstvih oddaljene za podano razdaljo. Tabela 'poznanstva' vsebuje
# množice indeksov znancev posameznih oseb.

def OsebeNaRazdalji(poznanstva, oseba, razdalja):

  # Postopamo zelo podobno kot pri funkciji 'Dosegljive'. Zanima nas
  # obogatitev množice 'dosegljive' v koraku 'razdalja'.

  dosegljive = set([])
  obogatitev = set([oseba])
  dosegljive.update(obogatitev)

  # Po prvem koraku sledeče zanke bo množica 'obogatitev' vsebovala znance
  # podane osebe, po drugem znance znancev podane osebe (a brez neposrednih
  # znancev) itd.
  for r in range(razdalja):

    # Izračunamo obogatitev v tem koraku (množica znancev oseb, s katerimi
    # smo množico 'dosegljive' obogatili v prejšnjem koraku, vendar brez
    # oseb, ki so v množici 'dosegljive' že prisotne).
    nove = set([])
    for ob in obogatitev:
      nove.update(poznanstva[ob])
    obogatitev = nove.difference(dosegljive)

    # Obogatimo množico 'dosegljive'.
    dosegljive.update(obogatitev)

  # Množica 'obogatitev' sedaj vsebuje iskane osebe.
  return obogatitev

prijateljstva = [set([1]), set([0, 5, 7]),
                 set([4, 5]), set([6]),
                 set([2]), set([1, 2, 7]),
                 set([3]), set([1, 5])]
for i in range(10):
  izpis = 'Osebe na razdalji ' + i
  osebe = OsebeNaRazdalji(poznanstva, 0, i)
  izpis += ' od osebe 0: ' + osebe
  print(izpis)

Primeri uporabe množice

Koda je nekoliko enostavnejša, če v vsakem koraku najprej izračunamo obogatitev množice dosegljive in množico šele potem dejansko obogatimo. V nasprotnem primeru bi namreč morali v glavni zanki vzdrževati dve množici (trenutno in tisto iz prejšnjega koraka) in obogatitev vsakokrat izračunati kot njuno razliko.

Če je razdalja enaka 0, naj vrnjena množica vsebuje samo podano osebo. Če je razdalja enaka 1, nas zanimajo vsi znanci podane osebe, vendar brez podane osebe same. V primeru razdalje 2 nas zanimajo znanci znancev podane osebe, vendar brez podane osebe same in njenih neposrednih znancev.
