5.

Napiši in preizkusi funkcijo OdsotneCrke(niz), ki vrne množico črk angleške abecede, ki se ne pojavijo v podanem nizu. Niz vsebuje samo presledke in male črke angleške abecede.

Razlika množic!

# Vrne množico črk angleške abecede, ki ne
# nastopajo v podanem nizu.
def OdsotneCrke(niz):

# izdelaj množico vseh črk
vseCrke = set([])
for koda in range(ord('a'), ord('z') + 1):
vseCrke.add(chr(koda))

return vseCrke.difference(set(niz))

print(OdsotneCrke('beseda'))
print(OdsotneCrke('fani bo psu zgradila hotel'))
print(OdsotneCrke('fox jumps over the dog'))

6.

Napiši in preizkusi funkcijo NapadenihPolj(n, polozajiDam), ki sprejme pozitivno celo število n in množico koordinat šahovskih dam na šahovnici n×n v obliki parov (vrstica, stolpec) in vrne število polj šahovnice, ki jih napada vsaj ena dama. Koordinate so v obeh smereh v razponu od 0 do n − 1. Dama napada vsa polja v isti vrstici, stolpcu in obeh diagonalah razen polja, na katerem stoji sama.

Na primer, klic funkcije NapadenihPolj(4, set([(0, 0)])) bi vrnil vrednost 9, saj dama v kotu šahovnice 4×4 napada 9 polj, klic NapadenihPolj(4, set([(0, 0), (1, 1)])) pa bi vrnil vrednost 14, saj ti dve dami napadata vsa polja razen polj (2, 3) in (3, 2).

Polja, ki jih napada dama, lahko predstaviš z množico parov.

Posamezne množice napadenih polj združi z unijo.

# Vrne število polj, ki jih napadajo dame na
# podanih položajih.
def NapadenihPolj(n, polozajiDam):
# Množico napadenih polj izdelaj kot unijo
# množic polj, ki jih napadajo posamezne
# dame.
napadenaPolja = set([])
for polozaj in polozajiDam:
napadenaPolja.update(Napadi(n, polozaj))
return len(napadenaPolja)

# Vrne množico polj, ki jih napada dama na
# podanem položaju.
def Napadi(n, polozajDame):
napadi = set([])
vrstica = polozajDame[0]
stolpec = polozajDame[1]

# V množico 'napadi' dodaj polja v isti
# vrstici.
for st in range(n):
napadi.add((vrstica, st))

# V množico 'napadi' dodaj polja v istem
# stolpcu.
for vr in range(n):
napadi.add((vr, stolpec))

# V množico 'napadi' dodaj polja na
# diagonali zgoraj levo - spodaj desno.
for i in range(-n, n):
vr = vrstica + i
st = stolpec + i
if vr >= 0 and vr < n and \
st >= 0 and st < n:
napadi.add((vr, st))

# V množico 'napadi' dodaj polja na
# diagonali spodaj levo - zgoraj desno.
for i in range(-n, n):
vr = vrstica - i
st = stolpec + i
if vr >= 0 and vr < n and \
st >= 0 and st < n:
napadi.add((vr, st))

# Iz množice 'napadi' odstrani polje, na
# katerem stoji dama, saj dama tega polja
# ne napada.
napadi.remove((vrstica, stolpec))

return napadi

print(NapadenihPolj(4, set([(0, 0)])))
print(NapadenihPolj(4, set([(0, 0), (1, 1)])))
print(NapadenihPolj(8, set([(4, 3)])))
print(NapadenihPolj(8, set([(2, 3), (4, 6)])))

7.

Napiši funkcijo IzberiElement(mnozica), ki vrne enega od elementov (ni važno, katerega) podane množice. Množica je podana kot spremenljivka pythonovega tipa set. Če je podana množica prazna, naj funkcija vrne None. (To je posebna pythonova beseda, ki označuje neveljavno vrednost.)

# Če je podana množica prazna, vrne None, sicer
# pa vrne enega od njenih elementov.
def IzberiElement(mnozica):
prviElement = None
for element in mnozica:
prviElement = element
break
return prviElement

print(IzberiElement(set([5, 10, 15, 20, 25])))
dnevi = set(['ponedeljek', 'torek', 'sreda',
'četrtek', 'petek', 'sobota',
'nedelja', 'ponedeljek'])
print(IzberiElement(dnevi))
print(IzberiElement(set([])))
