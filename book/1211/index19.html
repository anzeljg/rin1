Namig

(1) Sprehodi se po elementih množice in sproti posodabljaj spremenljivko, ki vsebuje največjega izmed do sedaj obravnavanih elementov.
(2) Upoštevaj, da so v množici samo pozitivna števila.

Rešitev

# Vrne največji element podane neprazne množice, pri čemer vemo, da množica
# vsebuje samo pozitivna cela števila.
def Najvecji(mnozica):

  # naj: doslej največji element
  naj = 0
  for element in mnozica:
    if element > naj:
      naj = element
  return naj

print(Najvecji({3, 5, 7, 2, 6, 4}))
print(Najvecji({10}))

Namig

Ker v množici ne moreš neposredno dobiti »prvega« elementa (zakaj?), si pomagaj s spremenljivko logičnega tipa, ki ima vrednost True samo v prvem obhodu zanke, s katero se sprehodiš po množici.

Rešitev

# Vrne največji element podane neprazne množice, pri čemer lahko množica
# vsebuje poljubna cela števila.
def Najvecji(mnozica):
  prvicVZanki = True
  naj = 0

  for element in mnozica:
    # Ko gremo prvič v zanko, ima spremenljivka 'prvicVZanki' vrednost True
    # (takrat nastavimo spremenljivko 'naj' na trenutni element množice),
    # sicer pa False (takrat posodobimo spremenljivko 'naj', če je manjša
    # od trenutnega elementa množice).
    if prvicVZanki or element > naj:
      prvicVZanki = False
      naj = element
  return naj

print(Najvecji({3, -5, -7, -2, 6, 4}))
print(Najvecji({-3, -5, -7, -2, -6, -4}))
print(Najvecji({-10}))

Namig

Si lahko pomagaš tako, da elemente tabele prepišeš v množico?

Rešitev

# Vrne število različnih elementov v tabeli t.
def SteviloRazlicnih(t):
  return len(set(t))

print(SteviloRazlicnih([5, 7, 2, 7, 5, 5, 7, 2]))
print(SteviloRazlicnih([10, 5, 20, 15, 0, -10]))
print(SteviloRazlicnih([3, 3, 3, 3, 3, 3, 3]))
print(SteviloRazlicnih([]))

Namig

Katera operacija nam vrne skupne elemente dveh množic?

Rešitev

# Vrne množico črk, ki nastopajo v obeh podanih nizih.
def SkupneCrke(niz1, niz2):
  # množica znakov niza 1
  znaki1 = set(niz1)

  # množica znakov niza 2
  znaki2 = set(niz2)

  # znaki, ki nastopajo v obeh nizih
  skupniZnaki = znaki1.intersection(znaki2)

  # odstranimo morebiten presledek
  return skupniZnaki.difference(set(' '))

print(SkupneCrke('marko skače', 'mirko skače'))
print(SkupneCrke('drevo', 'vedro'))
print(SkupneCrke('beseda', 'konj'))

Naloge

1

Napiši in preizkusi funkcijo Najvecji(mnozica), ki vrne največji element v podani neprazni množici. Lahko predpostaviš, da množica vsebuje samo pozitivna cela števila. Množica je podana kot spremenljivka pythonovega tipa set. Nalogo reši brez uporabe vgrajene pythonove funkcije max.

2

Enako kot naloga 1, le da lahko množica vsebuje poljubna cela števila.

3

Napiši in preizkusi funkcijo SteviloRazlicnih(t), ki vrne število različnih elementov v tabeli t.

4

Napiši in preizkusi funkcijo SkupneCrke(niz1, niz2), ki vrne množico črk, ki se pojavijo v obeh podanih nizih. Niza vsebujeta samo presledke in male črke angleške abecede.
