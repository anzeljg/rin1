*Implementacija z dvojiškim iskalnim drevesom

Dvojiško drevo lahko v pythonu predstavimo podobno kot povezani seznam: s tabelo vozlišč. Vozlišče na indeksu 0 (glava) je predstavljeno s tabelo, ki vsebuje število elementov in indeks korena drevesa, vsa ostala vozlišča pa so predstavljena s tabelami, ki vsebujejo element v vozlišču ter indeksa njegovega levega in desnega naslednika. Kot neveljaven indeks nam bo ponovno služilo število 0. Podobno kot pri povezanem seznamu so lahko vozlišča v tabeli nanizana v poljubnem vrstnem redu; pomembno je le, da so med seboj pravilno povezana z indeksi. Oglejmo si primer:


Gornja tabela je v pythonu predstavljena tako:
drevo = [[7, 1], [20, 2, 6], [10, 3, 4],
[7, 0, 0], [15, 5, 0], [12, 0, 0],
[30, 0, 7], [35, 0, 0]]
