Namig

Za vsak indeks i v tabeli se sprehodi po vseh indeksih j v tabeli (dvojna zanka!). Če je vsota elementov na trenutnih indeksih i in j enaka ciljni vsoti, izpiši trenutna indeksa.

Rešitev

# Program izpiše vse pare indeksov elementov v podani tabeli, katerih vsota
# je enaka številu ciljnaVsota.
tabela = [7, 5, 3, 11, 4, 5, 6, -1, 3]
ciljnaVsota = 10

for i in range(len(tabela)):
  for j in range(i, len(tabela)):
    if tabela[i] + tabela[j] == ciljnaVsota:
      print(i, j)

Namig

Če je tabela urejena, zadošča, če med seboj primerjaš sosednje elemente.

Namig

Postavi spremenljivko logičnega tipa (npr. vsiRazlicni) na True. V urejeni tabeli se sprehodi po vseh indeksih od prvega do predzadnjega in vsakokrat preveri, ali je element na trenutnem indeksu enak elementu na naslednjem indeksu. Če je, postavi spremenljivko vsiRazlicni na False. Na koncu preveri vrednost te spremenljivke.

Rešitev

# Program izpiše DA, če so vsi elementi med seboj različni, sicer pa
# izpiše NE.
tabela = [7, 5, 1, 8, 9, 6, 3] # izpiše: DA
#tabela = [7, 5, 1, 7, 9, 6, 3] # izpiše: NE
urejena = sorted(tabela)

vsiRazlicni = True
for i in range(len(urejena) - 1):
  if urejena[i] == urejena[i + 1]:
    vsiRazlicni = False
    break

if vsiRazlicni:
  print('DA')
else:
  print('NE')
