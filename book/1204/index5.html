Sprehod po elementih tabele

Naloga

Vzemi svinčnik in kos papirja ter po korakih »izvedi« program za izračun vsote tabele in program za iskanje največje vrednosti. Kakšne so vrednosti posameznih spremenljivk po posameznih korakih izvajanja?

Naloga

Napiši program, ki v podani tabeli poišče najmanjšo vrednost.

# Program izpiše najmanjši element v podani
# tabeli.
prodaja = [182, 250, 225, 315, 328, 160, 51, 36,
340, 280, 378, 320]

# Spremenljivko najProdaja postavimo na prvi
# element tabele.
najProdaja = prodaja[0]

for prodajaMesec in prodaja:
# Če je prodaja v trenutnem mesecu manjša
# od doslej najmanjše prodaje, posodobimo
# spremenljivko najProdaja.
if prodajaMesec < najProdaja:
najProdaja = prodajaMesec

print(najProdaja)

Našo Lizo še zanima, v koliko mesecih je prodala vsaj po 100 kg sladkarij. Tudi pri tej nalogi potrebujemo pomožno spremenljivko — števec za štetje elementov z vrednostjo najmanj 100. Števec na začetku postavimo na 0, nato pa se sprehodimo po elementih tabele. Če je trenutni element enak najmanj 100, števec povečamo za 1. Na koncu števec izpišemo: 













 
  



 


Napišimo še program, ki preveri, ali je vsaj en element v tabeli enak najmanj 100. Če to drži, naj program izpiše DA, sicer pa NE.











 
  



 


Naloge se lahko lotimo enako kot prejšnje, le da na koncu namesto izpisa števca preverimo, ali je njegova vrednost večja od 0. Lahko pa si namesto s števcem pomagamo s spremenljivko logičnega tipa. Spremenljivko (recimo ji vsajEden) na začetku postavimo na False, ko naletimo na element z vrednostjo najmanj 100, pa jo postavimo na True. Po koncu sprehoda preverimo vrednost spremenljivke:









  
  



 






























 

Naloga
  



      





Zakaj smo po stavku vsajEden = True dodali še stavek break? Bi program deloval tudi brez stavka break?
















Ko naletimo na element z vrednostjo najmanj 100, lahko sprehod prekinemo in takoj izpišemo DA. Program bi deloval tudi brez stavka break, bi pa v nekaterih primerih opravljal odvečno delo.













 

Naloga
  



      





Napiši program, ki izpiše DA, če so vsi elementi tabele enaki najmanj 100. V nasprotnem primeru naj program izpiše NE. Program preveri s čim več različnimi tabelami. (Ime tabele seveda lahko ostane prodaja, spreminjaj pa njeno dolžino in vrednosti elementov.)
















Pomagaj si s štetjem elementov ali pa s spremenljivko logičnega tipa po zgledu prejšnje naloge.























# Program izpiše DA, če so vsi elementi tabele


# enaki najmanj 100.


# V nasprotnem primeru izpiše NE.


prodaja = [182, 250, 225, 315, 328, 160, 51, 36,


340, 280, 378, 320]


vsi = True


for prodajaMesec in prodaja:


if prodajaMesec < 100:


vsi = False





if vsi:


print('DA')


else:


print('NE')
