Namig

Najprej poišči najmanjši element tabele.

Namig

Nato se sprehodi po indeksih tabele in vsakokrat preveri, ali je element na trenutnem indeksu enak najmanjšemu elementu tabele. Če je, izpiši trenutni indeks.

Rešitev

# Program izpiše indekse vseh najmanjših elementov v podani tabeli.
tabela = [30, 20, 20, 80, 20, 40]

# Določimo najmanjši element.
najmanjsi = tabela[0]
for element in tabela:
  if element < najmanjsi:
    najmanjsi = element

# Sprehodimo se po indeksih tabele in vsakokrat pogledamo, ali je element
# na trenutnem indeksu enak najmanjšemu elementu tabele. Če je, izpišemo
# trenutni indeks.
for indeks in range(len(tabela)):
  if tabela[indeks] == najmanjsi:
    print(indeks)

Sledenje programu: Indeks meseca z najnižjo prodajo

1    prodaja = [182, 250, 225, 315, 328, 160,
2               51, 36, 340, 280, 378, 320]
3    indeksMin = 0
4    for indeks in range(len(prodaja)):
5      if prodaja[indeks] < prodaja[indeksMin]:
6        indeksMin = indeks
7    print(indeksMin)

Pri sledenju gornjemu programu s pomočjo papirja in svinčnika, bo vrednost spremenljivke prodaja in izraza len(prodaja) vedno enaka oziroma se ne bo spreminjala. Zato te spremenljivke in tega izraza ne bomo zapisovali v tabelo.

Zapišimo torej v tabelo, kako se spreminjajo vrednosti spremenljivk in izrazov v gornjem programu. Naslovi stolpcev tabele bodo naslednji:

  • Korak predstavlja zaporedno številko koraka v sledenju programa
  • # predstavlja vrstico kode v kateri se zgodi sprememba vrednosti spremenljivke ali izraza
  • indeksMin predstavlja, kako se spreminjajo vrednosti te spremenljivke
  • indeks predstavlja, kako se spreminjajo vrednosti te spremenljivke
  • prodaja[indeks] < prodaja[indeksMin] predstavlja, kako se spreminjajo vrednosti tega izraza

Sprehod po indeksih tabele

Vaja

Po korakih »izvedi« zgornji program s svinčnikom in papirjem.

V razmislek

Program za iskanje indeksa najmanjšega elementa v tabeli bi deloval, tudi če bi se sprehodili od drugega elementa do zadnjega: 

tabela = [...]
indeksMin = 0
for indeks in range(1, len(tabela)):
  if tabela[indeks] < tabela[indeksMin]:
    indeksMin = indeks
print(indeksMin)
Zakaj?

Vaja

Napiši program, ki za podano tabelo izpiše indekse vseh najmanjših elementov (če jih je več). Na primer za tabelo [30, 20, 20, 80, 20, 40] naj program izpiše indekse 1, 2 in 4.

Spreminjanje, dodajanje in odstranjevanje elementov

Element na nekem indeksu v tabeli spremenimo s preprostim prirejanjem: 

>>> t = [1, 2, 3, 4]
>>> t[1] = 10
>>> t
[1, 10, 3, 4]

S stavkom tabela.append(element) dodamo element na konec tabele, s stavkom tabela.insert(indeks, element) pa element vstavimo pred element s podanim indeksom: 

>>> t = [10, 20, 30, 40]
>>> t.append(100)
>>> t
[10, 20, 30, 40, 100]
>>> t.insert(1, 60)
[10, 60, 20, 30, 40, 100]

Element lahko iz tabele odstranimo na dva načina. Ukaz 

tabela[indeks : indeks + 1] = []

odstrani element na indeksu indeks, ukaz 

tabela.remove(vrednost)

pa odstrani prvi element s podano vrednostjo. Če element ne obstaja, sproži izjemo.
