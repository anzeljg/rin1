Naloge

10.

Napiši program, ki v podani tabeli lihe dolžine s samimi različnimi elementi poišče element, ki se po velikosti uvršča točno na sredino. Na primer, za tabelo
tabela = [50, 20, 80, 90, 70, 60, 40]
bi program izpisal 60.

Nalogo najlažje rešiš tako, da s pomočjo funkcije sorted najprej izdelaš urejeno kopijo tabele.

# Program izpiše srednji element v podani
# tabeli ob predpostavki, da tabela vsebuje
# same različne elemente in da je število
# elementov liho.
tabela = [50, 20, 80, 90, 70, 60, 40]

# Izdelamo urejeno kopijo tabele.
urejena = sorted(tabela)

# Srednji element se v urejeni kopiji nahaja
# točno na sredini (npr. na indeksu 3, če je
# tabela dolga 7 elementov).
indeksSrednjega = len(urejena) // 2
print(urejena[indeksSrednjega])

11.

Otroci stojijo v ravni vrsti in se s pomočjo zunanjega »izštevalca« igrajo izštevanko. Izštevalec v vsakem krogu izštevanke šteje od 1 do stBesed in pri tem s prstom potuje po otrocih. Prične pri prvem otroku v vrsti (od tistih, ki so še ostali). Ko pride do konca vrste, ponovno nadaljuje pri prvem otroku. Igra se zaključi, ko ostane samo še en otrok.

Napiši program, ki za podano tabelo imen otrok in število stBesed izpiše imena otrok, ki izpadejo v posameznih krogih izštevanke. V primeru definicij
otroci = [’Ana’, ’Bojan’, ’Cvetka’, ’Dejan’, ’Eva’]
stBesed = 9
bi program izpisal sledeče:
Dejan
Ana
Eva
Bojan

Koliko krogov ima izštevanka?

V vsakem krogu najprej ugotovi, kdo izpade. Natančneje: določi indeks otroka, ki izpade. Si lahko morda pomagaš z ostankom pri deljenju?

Ko ugotoviš, kdo izpade, moraš element na tem indeksu samo še odstraniti iz tabele otroci.

# Program izpiše zaporedje imen otrok, ki
# izpadejo pri izštevanki.

# imena otrok
otroci = ['Ana', 'Bojan', 'Cvetka', 'Dejan',
'Eva']

# število besed izštevanke
stBesed = 9

# Izštevanka ima en krog manj, kot je otrok.
stKrogov = len(otroci) - 1

# stOtrokVIgri: število otrok, ki so še vedno
# v igri
stOtrokVIgri = len(otroci)

for krog in range(stKrogov):
# Določimo otroka, ki izpade v trenutnem
# krogu.
indeksIzpad = (stBesed - 1) % stOtrokVIgri

# Izpišemo ime izpadlega otroka.
print(otroci[indeksIzpad])

# Izločimo izpadlega otroka iz igre.
otroci[indeksIzpad : indeksIzpad + 1] = []

# Število otrok v igri se zmanjša za 1.
stOtrokVIgri -= 1

12.

Napiši program, ki izpiše dolžino najdaljšega strnjenega zaporedja ničel v podani tabeli. Na primer, za tabelo
tabela = [0, 6, 5, 0, 0, 0, 0, -1, 0, 0, 0, 10]
naj program izpiše število 4, za tabelo
tabela = [1, 6, 5, -1, 3, 7]
pa število 0.

Najprej reši lažji problem: po vsaki prebrani ničli izpiši dolžino trenutnega strnjenega zaporedja ničel. Vzdržuj spremenljivko, ki hrani to dolžino. Kaj moraš narediti, ko pri sprehodu po tabeli naletiš na ničlo, in kaj, ko naletiš na število, različno od 0?

# Program izpiše dolžino najdaljšega
# strnjenega zaporedja ničel v podani
# tabeli.
tabela = [0, 6, 5, 0, 0, 0, 0, -1, 0, 0, 0,
10, 5]

# dolžina trenutnega zaporedja ničel
dolzinaTreZaporedja = 0

# dolžina doslej najdaljšega zaporedja ničel
dolzinaNajZaporedja = 0

for element in tabela:
if element == 0:
# Prebrali smo ničlo; trenutno zaporedje
# ničel se podaljša.
dolzinaTreZaporedja += 1
else:
# Prebrali smo število, različno od 0.
# Trenutno zaporedje ničel je spet prazno.
dolzinaTreZaporedja = 0
# Če je trenutno zaporedje ničel daljše
# od doslej najdaljšega, posodobimo
# spremenljivko dolzinaNajZaporedja.
if dolzinaTreZaporedja > dolzinaNajZaporedja:
dolzinaNajZaporedja = dolzinaTreZaporedja

print(dolzinaNajZaporedja)
