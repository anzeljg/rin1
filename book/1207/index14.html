Naloge

1.

Zapiši definicijo problema, na katerega se nanašata obe sledeči vprašanji:
  • Prvi dijak je na testu zbral 93, drugi 75, tretji 85, četrti pa 91 točk. Za petico je treba zbrati več kot 89 točk. Koliko dijakov je dobilo petico?
  • Podane so cene šestih izdelkov: 37, 20, 28, 45, 40 in 32 EUR. Koliko izdelkov stane več kot 30 EUR?

Definicijo lahko zapišemo na različne načine, ena možnost pa je takšna:
  • Vhod: Tabela celih števil in celo število m (meja).
  • Pričakovani izhod: Število elementov tabele, ki so večji od m.

2.

Zapiši definicijo problema, ki mu pripadajo vsi sledeči primerki:
Vhod Pričakovani izhod
a = 2, b = 4 16
a = 3, b = 1 3
a = 5, b = 3 125
a = 10, b = 0 1
a = 10, b = 2 100

Ena od možnosti je takšna:
  • Vhod: Celi števili a in b, pri čemer velja a > 0 in b ≥ 0.
  • Pričakovani izhod: Vrednost potence ab.

3.

Kaj je narobe s sledečo definicijo problema?
  • Vhod: Tri poljubna cela števila.
  • Pričakovani izhod: Tisto vhodno število, ki ni niti najmanjše niti največje.
Definicijo ustrezno popravi. Popravi samo opredelitev vhoda, opredelitve pričakovanega izhoda pa ne spreminjaj.

Če sta vsaj dve števili enaki, potem je vsako število najmanjše ali pa največje. Zaradi tega moramo zahtevati, da so vhodna števila medsebojno različna.

4.

Sestavi algoritem za problem iz prejšnje naloge. Algoritem zapiši s programom, funkcijo in diagramom poteka.

Program:
# Prebere tri cela števila in izpiše srednje
# med njimi.
prvo = int(input('Vnesi prvo število: '))
drugo = int(input('Vnesi drugo število: '))
tretje = int(input('Vnesi tretje število: '))

# Določi manjše in večje izmed prvih dveh
# števil.
if prvo < drugo:
manjse12 = prvo
vecje12 = drugo
else:
manjse12 = drugo
vecje12 = prvo

# Poglej, kam sodi tretje število.
if tretje < manjse12:
srednje = manjse12
elif tretje > vecje12:
srednje = vecje12
else:
srednje = tretje

print(srednje)

Funkcija:
# Vrne srednje izmed podanih treh celih števil.
def srednje(prvo, drugo, tretje):
# Določi manjše in večje izmed prvih
# dveh števil.
if prvo < drugo:
manjse12 = prvo
vecje12 = drugo
else:
manjse12 = drugo
vecje12 = prvo

# Poglej, kam sodi tretje število.
if tretje < manjse12:
srednje = manjse12
elif tretje > vecje12:
srednje = vecje12
else:
srednje = tretje

return srednje

# Preizkus funkcije
print(srednje(10, 20, 30))
print(srednje(5, 7, 6))
print(srednje(4, 2, 5))
print(srednje(25, 27, 22))
print(srednje(-1, -5, -3))
print(srednje(10, 6, 3))

5.

Napiši algoritem, ki za podano neprazno tabelo celih števil preveri, ali so vsa števila v njej enaka. (Vhod v problem je neprazna tabela celih števil, pričakovani izhod pa je vrednost True, če so vsa števila enaka, oziroma False sicer.) Algoritem zapiši s funkcijo in v nedvoumnem naravnem jeziku.

Zapis algoritma s funkcijo:
# Vrne True, če so vsa števila v podani tabeli
# enaka. Sicer vrne False.
def vsaEnaka(tabela):
if len(tabela) == 1:
return True

for stevilo in tabela[1:]:
if stevilo != tabela[0]:
return False
return True

# Preizkus funkcije
print(vsaEnaka([7, 7, 7, 7]))
print(vsaEnaka([7, 7, 7, 6]))
print(vsaEnaka([10]))
print(vsaEnaka([1, 2, 3]))

Zapis algoritma v naravnem jeziku:
  1. Vhod: Neprazna tabela celih števil.
  2. Če je dolžina tabele enaka 1, je izhod enak True.
  3. V nasprotnem primeru za vsak element tabele (razen za prvega) preverimo, ali je enak prvemu elementu tabele. Če za katerega od elementov to ne drži, postopek takoj zaključimo, in to z izhodom False.
  4. Če pridemo do tega koraka, je izhod enak True.

6.

Prepiši sledeči algoritem iz psevdokode v pythonov program:
vhod: cela števila a, b in c
če velja a > b, potem:
zamenjaj vrstni red spremenljivk a in b
če velja b > c, potem:
zamenjaj vrstni red spremenljivk b in c
če velja a > c, potem:
zamenjaj vrstni red spremenljivk a in c
izhod: vrednosti spremenljivk a, b in c

Kakšen problem rešuje ta algoritem?

Algoritem rešuje problem naraščajočega urejanja treh števil.

# Program za naraščajoče urejanje treh števil.
a = int(input('Vnesi prvo število: '))
b = int(input('Vnesi drugo število: '))
c = int(input('Vnesi tretje število: '))

if a > b:
t = a
a = b
b = t
if b > c:
t = b
b = c
c = t
if a > b:
t = a
a = b
b = t

print(a, b, c)
