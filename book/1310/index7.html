Naloge

1.

Izberi vse načine reševanja vzporednosti, ki smo jih spoznali:

2.

Ogledali smo si primer v delikatesi, kjer smo imeli le eno vrsto in eno prodajalko. Poskusi odgovoriti na spodnja vprašanja:
  • Kako bi razvrščanje potekalo recimo na pošti, kjer je ena vrsta in več okenc?
  • Kako se dogovorijo, katera stranka gre h kateremu okencu?
  • Kaj pa se zgodi, če sta ravno dve okenci hkrati prosti?

Postavi se v položaj stranke in razmisli, ali je bolje imeti eno ali več čakalnih vrst, če imamo več okenc? Kaj bi te motilo pri eni čakalni vrsti? Ali bi dve vrsti rešili tvojo zadrego? Kaj pa še več vrst? Kaj bo kriterij razvrščanja v vrste? Najprej si zamisli, kaj želiš doseči, ter se nato odloči, kako boš to dosegel.

3.

# Mnozica sedezev v vrsti

# funkcija vrne mnozico prostih sedezev v vrsti
def prosti():
return(vrsta)

# funkcija iz mnozice prostih sedezev v vrsti
# izbrise sedez b
def kupi(b):
if b in vrsta:
vrsta.remove(b)
else:
print("Ta sedez je ze zaseden.")

kupi(4)

4.

Kako bi rešil problem hkratne prodaje kino vstopnic s pomočjo monitorjev?

V vseh primerih prodaje vstopnic predstavlja izziv izdelava atomične funkcije oziroma transakcije. Monitor je ena od tehnologij, ki zagotavlja prav to.
