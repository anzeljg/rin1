6.

Vrnimo se k eni od nalog iz tretje učne enote (Zanke), ki jo lahko zdaj s funkcijami rešimo lepše in elegantneje.

Napiši funkcijo ZanimivaStevila(N, k), ki izpiše vsa tista praštevila med 1 in N, ki imajo v desetiškem zapisu vsaj k števk enakih 1 in ki nastopajo kot z v kakšni pitagorejski trojici (x; y; z).

Svojo rešitev strukturiraj tako, da se za preverjanje praštevilskosti, štetje števk z vrednostjo 1 ipd. uporablja ločene funkcije, ki jih potem kliče glavna funkcija ZanimivaStevila.

def StEnic(n):
st = 0
while n > 0:
if n % 10 == 1: st += 1
n //= 10
return st

def JePrastevilo(n):
k = 2
while k * k <= n:
if n % k == 0: return False
k += 1
return True

def ObstajaTrojica(c):
a = 1; b = c
while a <= b:
while a * a + b * b > c * c: b -= 1
if a * a + b * b == c * c: return True
a += 1
return False

def ZanimivaStevila(N, k):
for n in range(2, N + 1):
if StEnic(n) == k and JePrastevilo(n) and \
ObstajaTrojica(n):
print(n)

#ZanimivaStevila(100000, 3)
ZanimivaStevila(10000, 3)

7.

Napiši funkcijo KolikokratNajmanjsi(t), ki dobi kot parameter tabelo t in vrne število, ki pove, kolikokrat se v tabeli t pojavi najmanjši element te tabele.
Primer: KolikokratNajmanjsi([10, 15, 8, 13, 15, 8, 15, 17]) mora vrniti 2 (najmanjši element dane tabele je 8, ki se v njej pojavi 2-krat).

def KolikokratNajmanjsi(t):
kolikokrat = 0
najmanjsi = min(t)
for element in t:
if element == najmanjsi:
kolikokrat += 1
return kolikokrat

tabela = [10, 15, 8, 13, 15, 8, 15, 17]
print(KolikokratNajmanjsi(tabela))

Možna pa je še krajša rešitev:
def KolikokratNajmanjsi(t):
return t.count(min(t))

tabela = [10, 15, 8, 13, 15, 8, 15, 17]
print(KolikokratNajmanjsi(tabela))

8.

Malo težja različica prejšnje naloge: tvoja funkcija naj naredi samo en prehod čez tabelo t in naj ne uporablja drugih funkcij, kot so min in podobne.

def KolikokratNajmanjsi(t):
kolikokrat = 0
for element in t:
if kolikokrat == 0 or element < najmanjsi:
# Trenutni element je manjši
# od vseh dosedanjih
najmanjsi = element
kolikokrat = 1
elif element == najmanjsi:
# Nova pojavitev najmanjšega
# doslej znanega elementa
kolikokrat += 1
return kolikokrat

tabela = [10, 15, 8, 13, 15, 8, 15, 17]
print(KolikokratNajmanjsi(tabela))
