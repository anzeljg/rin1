Rešitev

# Preberemo vhodni števili.
a = int(input('Vnesi prvo število: '))
b = int(input('Vnesi drugo število: '))

# Poskušamo ju deliti; če je delitelj enak 0,
# se sproži izjema, ki jo prestrežemo.
try:
  kolicnik = a // b
  print('Količnik znaša ' + str(kolicnik) + '.')
except ZeroDivisionError:
  print('Z ničlo žal ne morem deliti ...')

Namig

Namig

Koliko števil med 1 in n je deljivih s številom k? Ali si lahko pomagaš s celoštevilskim deljenjem?

Rešitev

# Preberemo število učencev.
n = int(input('Število učencev v razredu? '))

# Izračunamo število učencev s številkami,
# deljivimi z 2.
n2 = n // 2

# Izračunamo število učencev s številkami,
# deljivimi z 3.
n3 = n // 3

# Izračunamo število osmoljencev s številkami,
# deljivimi z 2 in 3, torej s 6.
n6 = n // 6

# Število osmoljencev je kar enako številu
# učencev s številkami, deljivimi s 6.
steviloOsmoljencev = n6

# Število srečnežev dobimo tako, da od števila
# vseh učencev odštejemo tiste, katerih
# številke so deljive z 2 in 3. Vendar pa smo
# pri tem dvakrat odšteli tiste, katerih
# številke so deljive s 6, zato jih moramo
# enkrat prišteti.
steviloSrecnezev = n - n2 - n3 + n6

# Rezultata samo še izpišemo.
print('Število osmoljencev: ' + str(steviloOsmoljencev))
print('Število srečnežev: ' + str(steviloSrecnezev))

Naloge

8

Napiši program, ki prebere dve celi števili in izpiše njun količnik pri celoštevilskem deljenju. Če uporabnik za delitelj vnese ničlo, naj program izpiše ustrezno sporočilo. Na primer: 

Vnesi prvo število: 15
Vnesi drugo število: 4
Količnik znaša 3.


Vnesi prvo število: 15
Vnesi drugo število: 0
Z ničlo žal ne morem deliti ...

Nalogo reši tako, da prestrežeš izjemo tipa ZeroDivisionError, ki se sproži ob deljenju z ničlo.

9

Učiteljica pride v razred z n učenci, učencem razdeli listke s številkami od 1 do n (vsak dobi svojo številko) in pravi takole: »Tisti, katerih številka je deljiva z 2, boste danes čistili okolico šole. Tisti, katerih številka je deljiva s 3, boste okolico čistili jutri. Ostali ste oba dneva prosti.« Napiši program, ki prebere število n in izpiše

  • število osmoljencev, ki bodo okolico čistili oba dneva;
  • število srečnežev, ki bodo oba dneva prosti.

Primer: 

Število učencev v razredu? 20
Število osmoljencev: 3
Število srečnežev: 7
